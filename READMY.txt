Программа на языке PYTHON для работы с GOOGLE таблицами.
Для начала установите все необходимые библиотеки при помощи команды pip install -r libraries.txt
Чтобы программа работала корректно авторизуйтесь и создайте подключение API в GOOGLE таблицах с возможностью доступа и редактирования по ссылке.
Получите JSON файл ключ для доступа к вашим таблицам, создайте файл TOKEN.json и скопируйте все содержимое вашего JSON ключа в него.
Файл TOKEN.json должен быть в одной директории с проектом или укажите полный путь к этому файлу.
В файле constants.py в константе spreadsheet_id укажите вашу уникальную ссылку для доступа в GOOGLE sheets.
Если вы все сделали верно можно запускать файл main.py и программа запуститься корректно.

ОПИСАНИЕ САМОЙ ПРОГРАММЫ

Программа спросит у пользователя есть у него или нет таблицы для вывода результатов проверки или нет. Если нет то предложит их создать.
После того как таблицы созданы программа предложит запустить проверку и проверит исходную таблицу и выявит где были внесены корректировки.
